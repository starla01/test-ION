## Scripts disponibles

Para ejecutar el proyecto puede corres el siquiente comando en la terminal:

 ```sh
 yarn start
 ```
 
 ó
 
 ```sh
npm run start
 ```
 
Abrir [http://localhost:3000](http://localhost:3000) para verlo en el navegador.
